# PHP MVC Learning Labs

A learning repository for PHP MVC, progressing from fundamentals to controllers, views, assets, and models.

## Sessions
- pertemuan1_pengenalan: MVC introduction and basic structure
- pertemuan2_controler: controllers
- pertemuan3_view: views and templates
- pertemuan4_asset: assets and public resources (CSS/JS/images)
- pertemuan5_model: models and data access

## Structure
Each session folder contains:
- app: core MVC files (controllers, models, views, core classes)
- public: entry point and public assets

## Requirements
- PHP 8.x (XAMPP recommended)
- Apache web server

## Running (XAMPP)
1. Start Apache in XAMPP.
2. Open a session in your browser, for example:
   - http://localhost/phpmvc/pertemuan1_pengenalan/public/
   - http://localhost/phpmvc/pertemuan3_view/public/

## Notes
- Some sessions include assets copied from Bootstrap for learning purposes.
- Each session is self-contained; use the public index as the entry point.
